# Першочергові налаштування
1. Сворюємо віртуальне середовище:
```
python -m venv venv
```
2. Активуємо віртуальне середовище:
```
source venv/bin/activate
```
3. Проінсталюйте залежності:
```
pip install -r task_1/requirements.txt 
```
